Understanding Hyperlinks in HTML

1. Basic Hyperlink using <a> and href

The <a> tag is used to create hyperlinks. The href attribute specifies the destination URL.

Example:

Visit Google

2. Using target="_blank"

The target attribute specifies where to open the linked document.

target="_blank" opens the link in a new tab.

Open Wikipedia in New Tab

3. Using target="_self"

target="_self" opens the link in the same tab (default behavior).

Open Example in Same Tab

4. Email Hyperlink

You can also create an email link using mailto: inside href.

Send an Email

5. Internal Page Link

Hyperlinks can also jump to a section within the same page using an id.

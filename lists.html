Understanding HTML Lists

1. Ordered List

An ordered list displays items in a numbered format.

  1. Wake up
  2. Brush teeth
  3. Have breakfast
  4. Go to school

2. Unordered List

An unordered list displays items with bullet points.

  • Apple
  • Banana
  • Mango
  • Orange

3. Nested List

A nested list is a list inside another list.

  • Computer
    • Monitor
    • Keyboard
    • Mouse
  • Mobile Phone
    • Screen
    • Battery

4. Ordered List inside an Unordered List

An ordered list can be placed inside an unordered list.

  • Steps to Make Tea:
    1. Boil water
    2. Add tea leaves
    3. Add milk and sugar
    4. Strain and serve
  • Steps to Make Coffee:
    1. Boil milk
    2. Add coffee powder
    3. Add sugar
    4. Stir well and serve

5. Definition List

A definition list is used to define terms and their descriptions.

HTML
HyperText Markup Language - used to create web pages.
CSS
Cascading Style Sheets - used to style web pages.
JavaScript
A programming language used to make web pages interactive.
